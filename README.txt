VIHAANBYTES OFFICIAL WEBSITE
=============================

This is a futuristic, long-scroll, fully responsive personal brand website
for "VihaanBytes Official" - a tech creator platform.

WEBSITE STRUCTURE:
1. index.html - Main HTML structure
2. style.css - All styling with cyber/neon theme
3. script.js - All JavaScript functionality
4. image.png - Logo/profile image (to be added by you)

IMPORTANT:
- Place your logo/image file as "image.png" in the same folder

HOW TO USE:
1. Place all files in the same folder
2. Add your logo as "image.png"
3. Open index.html in any modern browser
4. The website is fully functional

TECHNOLOGY USED:
- Pure HTML, CSS, JavaScript (no external libraries)
- CSS animations and keyframes
- Canvas API for particle effects
- Flexbox and CSS Grid for layouts
- Font Awesome icons
- Google Fonts (Orbitron, Exo 2)

CUSTOMIZATION:

- Add more content to chapter notes
- Change social links in HTML
